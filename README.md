# RickAndMdroy
Приложение состоит из двух экранов. На первом экране список персонажей, а на втором дополнительная информация о персонаже и список эпизодов, в которых встречался персонаж.
Использовал:

Hilt

Retrofit

Coroutines

Glide

Paging (RemoteMediator)

Room

Navigation
